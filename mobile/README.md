# Ecosistema SMAT - Monorepo


## Requisitos
- Python 3.10+
- Flutter 3.x
- Git

## Ejecución del Backend
1. Entrar a `backend/`
2. Instalar: `pip install -r requirements.txt`
3. Correr: `fastapi dev app/main.py --host 0.0.0.0`

## Ejecución del Móvil
1. Entrar a `mobile/`
2. Configurar la IP en `api_service.dart`.
3. Correr: `flutter run`
